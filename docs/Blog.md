L'objectif de ce projet est de proposer un assistant capable de planifier des entraînements de manière autonome,
plutôt que de donner une réponse fixe, ce projet s'adapte aux besoins changeants de l'utilisateur (niveau, matériel disponible et lieu d'entraînement)
...


Installation et Lancement:
Ouvrez votre terminal et exécutez la commande suivante :
pip install streamlit pandas
Une fois les bibliothèques installées, placez-vous à la racine du dossier du projet et lancez cette commande:
streamlit run src/app.py
